## quick start

- clone this repo
- install hugo (https://gohugo.io/)

```bash

# you may want to set HTTP_PROXY and HTTPS_PROXY depending on your environment
# to download go modules

# all example is in this folder
cd exampleSite

# preview
hugo server -b <your-access-url> --bind 0.0.0.0

# build
hugo
```
